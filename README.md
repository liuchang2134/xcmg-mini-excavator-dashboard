# XCMG 3.5t Mini Excavator Competitive Dashboard

This repository hosts the executive dashboard for the 3.5-ton mini excavator competitive analysis.

## Deliverables

- `index.html` - interactive executive dashboard
- `小挖常见工况竞品量化评分.xlsx` - quantitative scoring workbook
- `数据异常检查报告.xlsx` - data quality issue report

## Notes

The dashboard is based on the provided 3-4.xlsx source data. Product naming, missing-value handling, condition weights, and confidence levels have been standardized before publishing.
